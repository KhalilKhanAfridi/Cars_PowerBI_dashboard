# Cars_PowerBI_dashboard
The Pakistani used car market is massive — but navigating it without data is just guesswork.

I built an interactive Cars Data Analysis Dashboard to change that.

Some of what the data revealed:

— Toyota dominates the market, but Hybrid vehicles are quietly growing at 9%
— Automatic transmissions have overtaken manual for the first time
— Car prices have risen sharply post-2015, with the average now sitting at 3.36M
— Where you buy matters just as much as what you buy — pricing shifts significantly by city

The dashboard lets you filter by city, brand, fuel type, transmission, and year — so the insights adapt to whatever question you're asking.

In a market where most buying and selling decisions are still made on instinct, having clarity like this isn't just useful — it's an advantage.
